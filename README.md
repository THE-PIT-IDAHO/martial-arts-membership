# Martial Arts Membership Software (Reboot)

Custom membership + operations software for a martial-arts gym (The PIT style):

- Members, memberships, classes, and attendance
- Kiosk mode for front-desk / check-in
- Admin dashboard with reports (attendance, payments, tasks)
- Communication tools, promos, and more as we build it out

## Tech Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- TypeScript
- Tailwind CSS
- Prisma ORM + PostgreSQL
- JWT-based auth (custom)
